**Description:** Checks the difference in y value (height) between the target entity and the caster.

---

**Attributes:**

| Attribute | Aliases        | Description               |
| --------- | -------------  | ------------------------- |
| yDiff|  | The y difference to check |

---

**Examples:**

```
  TargetConditions:
  - yDiff{diff=>5} true
```

---

**Extra Information:**

- [x] Type: Entity
