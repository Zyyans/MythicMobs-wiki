**Description:** This condition tests if the target is within the given list of biomes.

---

**Attributes:**

| Attribute | Alias | Description              |
| --------- | ----- | ------------------------ |
| biome     | b     | A list of biomes to check|

---

**Examples:**

```
Conditions:
- biome DESERT,DESERT_HILLS
```
