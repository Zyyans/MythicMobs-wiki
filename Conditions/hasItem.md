**Description:** Tests if the target player has the given number of given material

**Type:** Entity

---

**Attributes:**

| Attribute | Alias   | Description |
| --------- | ------- | ----------- |
| item      | i    | A material to check for |
| amount    | none | A Number to check for |

---

**Examples:**

```
Conditions:
- hasitem{i=STICK;amount=2} true
```
