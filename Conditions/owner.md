**Description:** Checks if the target entity is the owner of the caster. 

The mob's owner must be set using the Set Owner mechanic if it's not a wolf.

**Type:** Compare

---

**Attributes:**

| Attribute | Alias | Description |
| --------- | ----- | ----------- |
| none      | none  | none        |

---

**Examples:**

```
TargetConditions:
- owner{} true
```
