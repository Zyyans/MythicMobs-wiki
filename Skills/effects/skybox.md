**Description:** 

Changes the skybox of the target.

---

**Attributes:**

| Attribute        | Alias | Description                                                   |
| ---------------- | ----- | ------------------------------------------------------------- |
| skybox           | sky, s, type, t, environment, env, e | What skybox should be shown to the player | 

---

**Examples:**

Makes the target see the end skybox

```
- skybox{s=END} @PIR{r=20}
```
